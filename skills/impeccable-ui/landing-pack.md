# Landing Pack

## objective
Create landing pages that communicate value fast, prove credibility, and drive one dominant conversion path.

## hierarchy model
1. Hero promise (audience + value + outcome).
2. Proof band (logos, metrics, testimonials with source cues).
3. Value mechanics (how it works in 3 steps or fewer).
4. Objection handling (risk reversal, FAQ, trust details).
5. Conversion close (final CTA with reinforcement).

## typography roles
- hero-heading: primary promise statement.
- hero-subcopy: concise elaboration of value.
- section-heading: thematic anchors for each narrative band.
- proof-metric: quantified credibility points.
- body-copy: supporting detail and objection handling.

## spacing rhythm
- Base rhythm token: 8px.
- Hero vertical breathing room: 64px top/bottom minimum.
- Section-to-section spacing: 48px minimum.
- Intra-section content spacing: 16px to 24px.
- CTA group spacing: 12px between primary and secondary actions.

## CTA strategy
- One primary CTA repeated at critical narrative checkpoints.
- CTA text is outcome-led ("Start free trial", "Book a demo").
- Secondary CTA offers lower commitment without visual conflict.
- Final CTA repeats core value proposition adjacent to action.

## anti-pattern watchlist
- Vague hero copy with no audience or measurable outcome.
- CTA label mismatch across sections.
- Social proof without attribution or concrete details.
- Feature dumping before value framing.
- Decorative visuals that compete with primary message.

## done criteria
- Core value proposition is understood within 3 seconds.
- One dominant conversion path is visually and semantically clear.
- Social proof includes concrete, attributable signals.
- Section order supports a coherent persuasion narrative.
- CTA language and visual priority are consistent from hero to close.
