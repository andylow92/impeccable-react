# Generic vs Impeccable Example

## Generic version (intentionally bad)
- Same radius and shadow on every box.
- Weak contrast and non-committal hierarchy.
- Progress shown as decorative percentage fill.

## Impeccable version
- Distinct structural zones with clear scan path.
- Contrast and color tied to decisions and risk.
- Progress represented as accountable execution states.

## Why one feels generic
Generic UI repeats the same container treatment and avoids visual decisions.

## Why one feels intentional
Intentional UI assigns visual weight by role: context, decision metric, risk, and action.
